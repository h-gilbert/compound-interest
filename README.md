# Compound Interest Calculator

A web-based compound interest calculator with interactive charts for visualizing investment growth over time.

## Features

- Calculate compound interest with customizable parameters
- Interactive charts powered by Chart.js
- Detailed breakdown of compound growth
- Fully client-side (no backend required)

## Usage

### Local Development

Run a local server:

```bash
python3 -m http.server 8002
```

Then open http://localhost:8002 in your browser.

### Routes

- `/` - Landing page
- `/calculator/` - Calculator application

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Chart.js 4.4 (via CDN)
